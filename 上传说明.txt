            快速上传指南
===============================================

方法 1：GitHub Desktop（推荐新手）
------------------------------------------
1. 下载安装：https://desktop.github.com/
2. 打开 GitHub Desktop → File → Add Local Repository
3. 选择：D:\code\story\novascribe-github
4. 点击 "Publish repository" 按钮
5. 完成！

方法 2：命令行（推荐开发者）
------------------------------------------
cd D:\code\story\novascribe-github
git init
git add .
git commit -m "Initial commit: NovaScribe v1.0.0"

# 在 GitHub 创建仓库后：
git remote add origin https://github.com/你的用户名/novascribe.git
git push -u origin main
